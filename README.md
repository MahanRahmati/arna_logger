## Introduction

A pretty logger for your applications.

## Getting Started

Add Arna Logger as a dependency in your pubspec.yaml

```yaml
dependencies:
  arna_logger: ^1.0.0
```

And import it

```dart
import 'package:arna_logger/arna_logger.dart';
```
